Connect your Bosch eBike

Before adding your bike, you need an authorization code from Bosch. Here's how:

1
Go to Bosch eBike app settings → tap Copy Login URL
2
Paste the URL into a desktop browser and sign in with your Bosch account
3
Open Developer Tools → Network tab (F12) before clicking login. After signing in, find the request starting with onebikeapp-ios:// and copy the value after code=
⏱ The code expires in about 60 seconds — complete step 3 quickly after signing in.
