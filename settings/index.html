Bosch eBike — Setup

Follow these steps each time you add a new bike or re-authenticate an existing one.

1 Generate a login URL
Generating login URL...
2 Sign in on a desktop browser
Paste the URL into a desktop browser — not your phone, as it will open the Flow app instead. Sign in with your Bosch SingleKey ID.
Open Developer Tools → Network tab (F12) before clicking login. After signing in, find the request starting with onebikeapp-ios:// and copy the value after code=. The code expires in ~60 seconds.
3 Use the code

Adding a new bike: Go to Devices → + → Bosch eBike and paste the code into the Authorization Code field.

Re-authenticating: Tap your existing device → ⚙️ Settings → paste the code into New Authorization Code → Save.
