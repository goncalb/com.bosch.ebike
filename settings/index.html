Bosch eBike — Setup

Follow these steps to add a new bike or re-authenticate an existing one.

⚠️ Desktop browser required. The login step must be completed on a computer — Chrome or Firefox. It cannot be done on a phone as the Bosch login redirects to a mobile deep link that only a desktop browser can intercept.
1 Generate your login URL

A unique secure login URL is generated below. Copy it and paste it into your desktop browser.

Generating login URL...

Each URL is single-use. If you need to try again, tap "Generate new URL".

2 Sign in on your desktop browser
  1. On your desktop computer, open Chrome or Firefox
  2. Press F12 to open Developer Tools → click the Network tab
  3. Paste the Login URL into the address bar and press Enter
  4. Sign in with your Bosch / One Bike App account
  5. After login, the browser will show an error — this is normal
  6. In the Network tab, find the request starting with onebikeapp-ios://
  7. Click it and copy the value after code= (everything before the next &)
⏱ The authorization code expires in about 60 seconds. Have Homey ready before signing in.
3 Paste the code into Homey

Adding a new bike:
Go to Devices → + → Bosch eBike and paste the code into the Authorization Code field.

Re-authenticating an existing bike:
Open the device → Settings → Advanced Settings → Connection → paste the code into Authorization Code → tap Save.
